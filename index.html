Pergunta 01

Qual das quatro etapas custou mais caro para o software neste mês?

As etapas mais caras foram Desenvolvimento e Suporte Técnico, empatadas com R$ 4.000,00 cada uma — juntas representam 57% do custo total do mês.

Desenvolvimento: R$ 4.000,00  |  Suporte Técnico: R$ 4.000,00
Pergunta 02

Se o código fosse lançado com menos bugs, qual seria o impacto financeiro direto?

Lançar o app com menos bugs afeta diretamente duas etapas:

Testes (QA)

O direcionador dessa etapa é a quantidade de bugs encontrados. Com menos bugs, o consumo cai e o custo alocado diminui. Cada bug a menos equivale a R$ 200,00 a menos no custo.

Suporte Técnico

Menos bugs geram menos problemas para os usuários, o que reduz o número de chamados abertos. O custo do Suporte cai junto — cada chamado evitado poupa R$ 50,00.
